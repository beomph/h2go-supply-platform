# Data

이 폴더는 H2GO AI 고도화 작업을 위한 **샘플/시뮬레이션 데이터**를 보관합니다.

## 샘플 데이터

- `data/sample/stations.json`: 충전소(수요처) 목록
- `data/sample/sites.json`: 출하센터/생산지 목록
- `data/sample/trailers.json`: T/T(튜브트레일러) 자원
- `data/sample/drivers.json`: 운송기사 자원
- `data/sample/travel_matrix.json`: 이동 시간/거리 매트릭스(샘플)
- `data/sample/demand_timeseries.csv`: 수요(판매) 시계열(샘플)
- `data/sample/sensor_timeseries.csv`: 센서/운영 시계열(이상 탐지 샘플)

스키마 정의는 `.taskmaster/docs/data_schema.md`를 참고하세요.
